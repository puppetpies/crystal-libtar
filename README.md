# crystal-libtar

Libtar bindings for the Crystal language

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  crystal-libtar:
    github: puppetpies/crystal-libtar
```


## Usage


```crystal
require "crystal-libtar"
```
